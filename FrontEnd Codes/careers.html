Front End Developer

(Senior Level)

Job Description

We're looking for a Front End Developer to craft responsive, accessible, and high-performing web and mobile interfaces. You'll work closely with our design and backend teams to turn ideas into engaging user experiences.

Requirements:

  • Strong skills in HTML, CSS, and JavaScript
  • Experience with front-end frameworks (React, Vue, or Angular)
  • Understanding of responsive and mobile-first design
  • Familiarity with version control (Git)

Back End Developer

(Senior Level)

Job Description

We're looking for a Backend Developer to build and maintain the server-side of our web and mobile applications. You'll work with our team to create secure, scalable, and high-performance solutions.

Requirements:

  • Proficiency in (Node.js / Python / PHP / Java)
  • Experience with databases (MySQL, PostgreSQL, or MongoDB)
  • Knowledge of REST APIs and Git
  • Problem-solving mindset

Full Stack Developer

(Senior Level)

Job Description

We're looking for a Full Stack Developer to build and maintain both the front-end and back-end of our weband mobile applications. You'll work across the stack to create secure, scalable, and seamless digital experiences.

Requirements:

  • Proficiency in (JavaScript, Node.js, React, or similar frameworks)
  • Experience with databases (MySQL, PostgreSQL, or MongoDB)
  • Knowledge of REST APIs, Git, and responsive design
  • Problem-solving mindset
